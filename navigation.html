Application of design thinking methodology

Improving navigation on publications (2022)

Duration of project : 1.5 month

  • - User need : "I want to have a more cohesive way to read my long publication"
  • - Problem statement : "Having an infinite scroll to access to any page of my publication (>100 pages) can be complicated"

    • Steps done :
  • - User research : Primary & secondary sources (interviews and state of the art)
  • - Prototyping : Worked alongside design team to provide a first prototype to iterate with a selection of users
  • - Usability testing : Tested several versions of prototypes and selection of one. Out-loud techniques+script + missions to do.
  • - Developping : Creation of specifications and JIRA tasks
  • - Testing : Technical testing & user feedbacks
